# **Result-Management-Application**
## **Setup Guide Instruction**

#### *Backend*
1. Open the project in Visual Studio Code.
2. Go to toolbar -> terminal -> Open new terminal.
3. Run "npm install" and "npm install --save nodemon" .
4. To run the project -> "nodemon app.js" or "npm start" on terminal.

#### *Frontend*
1. Open the URL "https://localhost:5000" on a browser.
